## @font-face
Download font file to your local storage
```css
@font-face {
    src: url("typefaces/jumbo.tiff");
    font-family: "jumbo"; /* font-family is the name that will be used to identify the font */
}
```

## Background Image
```css 
background-image {
    src: url("images/myBackground.jpg");
}
```
